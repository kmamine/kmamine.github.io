# Mohamed Amine Kerkouri - Personal Website

Academic website and blog built with Jekyll for GitHub Pages.

## About

Research Scientist specializing in:
- Computer Vision
- Visual Attention Prediction
- Medical Imaging
- Deep Learning

## Local Development

```bash
# Install dependencies
bundle install

# Run local server
bundle exec jekyll serve

# Build for production
bundle exec jekyll build
```

## Structure

```
├── _layouts/          # Page templates
├── _includes/         # Reusable components
├── _posts/            # Blog posts
├── assets/            # CSS, JS, images
├── index.html         # Homepage
├── research.html      # Research page
├── publications.html  # Publications list
├── blog.html          # Blog listing
├── cv.html            # CV page
└── contact.html       # Contact information
```

## Deploy

This site is configured for GitHub Pages. Push to the `main` branch to deploy automatically.
